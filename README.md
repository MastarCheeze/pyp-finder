# CAIE Past Year Paper Finder 🔍

🌐 Website: https://pypfinder.mastarcheeze.com

## Synopsis

This website finds past CAIE exam papers (IGCSE, O Levels, A Levels and Pre-U) using their paper code. It finds a link to the PDF file of the paper, along with other related resources like inserts, examiner reports and grade thresholds.

I made this because I got tired of having to manually search for exam papers in a search engine and clicking
through a dozen links before getting the PDF I want.

## API

The website uses an API to find exam papers. The API will be available for public use soon.
